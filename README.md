## GitHub Pages

This project is ready for GitHub Pages because it uses only static files.

### Recommended setup

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Push to the `main` branch.
4. Open repository `Settings`.
5. Go to `Pages`.
6. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
7. Save and wait for GitHub Pages to publish.

Your site URL will look like:

`https://your-username.github.io/your-repo-name/`

## Notes

- `index.html` is the homepage.
- `.nojekyll` is included for smooth static hosting on GitHub Pages.
- Before making the site public, replace the placeholder contact email in `legal.html`.
